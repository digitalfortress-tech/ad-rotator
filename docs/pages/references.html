1. ‚ú® API

There is a single method destroy() exposed by Ad-rotator

JS 
/* Setup typeahead */
const instance = typeahead({ /* config options */ });

/* Destroy typeahead */
instance.destroy();

The destroy() method destroys the typeahead instance, removes all event handlers and cleans up the DOM. It can be used when you want to deactivate autocompletion completely.
